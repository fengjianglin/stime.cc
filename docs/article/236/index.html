言论自由和名誉防护（十三）

《麦迪逊前提》

新生的美国政坛分裂成为两大阵营，这几乎是从刚建国开始就在发生的事情。立宪会议后，首任总统的职责当然毫无异议的落在了华盛顿肩上，这位开国将军顺利连任两届之后，情况逐渐变得复杂起来。在华盛顿执政期间，围绕着州权和联邦权孰轻孰重的问题，政坛要员们开始分化成为两大对立的政治派别，分别是主张合众国至上的联邦党人（Fedralist）和坚持州权优先的民主共和党人（DemocraticRepublicans）。两大派别分别集结在副总统亚当斯和国务卿杰佛逊麾下，渐渐势同水火。

民主共和党和联邦党之争

[Source: tes.com]

1796年，华盛顿届满卸任，亚当斯和杰佛逊随即开始了总统大位之争。最终联邦党人险胜，亚当斯以71对66票的结果竞选成功，杰弗逊则成为副总统。党争继续升温，共和党谴责联邦党的集权思想是要把丑恶的英国王室专制传统强加到人民头上，联邦党则嘲笑共和党人追求所谓个人权力是法国雅各宾派的再现。两党立下标靶，英国和法国纷纷躺枪，而很快一件事情的出现更是大大激化了两党恶斗的紧张气氛。这就是“XYZ事件”。

1797年6月，美国为了缓解因法国大革命而渐渐闹僵的美法两国关系派出使团访法。没想到，通过非正式管道接洽到的法国外交部长塔列兰（Tellerrand）居然向美国使团索贿作为开展谈判的前提。其实这样的做法在当时的欧陆各国也不算稀罕事，但是美国使团却觉得受到了冒犯。两名联邦党的谈判代表要求不再进行暗线联系，而共和党的代表盖里（ElbridgeGerry）却仍希望沟通能继续下去。老练的塔列兰从中离间，拉一派打一派，利用各种虚假消息使团出现裂隙，最后联邦党代表愤而回国。事情传到国内，亚当斯向国会说明了美方使团分裂的情形，但为了避免进一步激怒议员，他刻意按下了法国人索贿的情节没说。此时，国会内主战的联邦党和反战的共和党已经完全没有互信，使团分裂的事件更加重了这一危机。共和党人并不了解亚当斯是因为不希望扩大战争风险故而对有些事秘而不宣的苦心，和联邦党人一起向总统逼宫，要求展示整个事件的完整内情。无奈之下，亚当斯只好招来共和党人向他们透露经过。为了避免对号入座，亚当斯把事件涉及到的法国三位重要人物的姓名只以代号“X”、“Y”、“Z”代替，共和党人此时方知亚当斯的持重之心，但是事已如此，美国本已高涨的仇法情绪已经难以遏制，双方差一点走到战争的边缘。XYZ事件因此成为美国党争的最高潮。

美法外交“XYZ事件”

[Source: charlescrawford.biz]

这种环境下，人人亢奋，就连第一夫人阿比盖尔·亚当斯都声称，民主共和党正在全国散播“邪恶、无神论、腐败和造谣滋事的种子”。越演越烈的对抗中，控制国会的联邦党人为了打击日益蔓延的在野势力，铲除那些似乎遍地开花的种种反政府言论，最终出台了大杀器《防治煽乱法》。当时美国的法律在对待出版自由的问题上，基本延续了布莱克斯通爵士在《英国法律评论》中的经典态度，即事前自由，事后严查。然而从司法实务的角度来看，根本没有人把可能的追责当回事。这或许仍是在独立战争时期培养出来的任性恣意的文风使然，无限的夸大和尖刻的言辞大行其道，就连华盛顿也不能自外。华盛顿被费城《曙光报》形容为“我国一切不幸的源头”，“政治邪恶和腐败将会伴随着他而退出历史舞台”。更有甚者，他的头像还被画在驴子的身上。以华盛顿的威望之高，尚且遭到如此羞辱，当时的诋毁风气可见一般。很难说这一切都是共和党的背后挑唆所致，但是这种毫无节制的言论环境绝对极大的助长了党派之争的气焰。7月4日，联邦党优势下的国会正式通过《防治煽乱法》草案，这个精心选择的建国日的通过日期足以说明该法案背后的高度政治性。

《防治煽乱法》规定凡是以不实之词恶意诋毁和攻击总统、国会或者联邦议员，煽动民众对抗政府的言论或者出版物，最高可以判处两年徒刑，并处2000美元以下的罚金。2000美元在十八世纪末可是一笔不得了的巨款，后来有许多人因为交不起罚金而被不断的延押。不过，有一点很值得玩味，这部法律仅限定总统、议员为保护对象，副总统则不再其列，至于原因嘛很简单，副总统杰佛逊是共和党的人。

《防治煽乱法》中提出了一条准则，叫做不实之词的攻击，也就是说言论的内容如果属实，是可以构成对被告人的防护的。这看上去和稍早前的英国普通法有所不同。在普通法中，真实性不构成诽谤的抗辩基点，你说的话即使是真的，只要你是在抨击政府、王室或者宗教，你仍是有罪的。美国的1798年《防治煽乱法》似乎在这点上有了进步，然而在实际层面，这个抗辩点形同虚设，因为它要求被告人承担举证责任，自我证明所发表言论的正确性。这在很多时候是难以做到的，比如对一些预测性的或想象性的评论，能用什么证据来证明呢？而且当时在陪审团的选择过程中也有很多的暗箱操作，法院经常安排大量的亲联邦党的人参与陪审，所以被告即使给出了证据，也很难被认定为有效力。在后来，大量的冤假错案因此发生。

当时的一副讽刺漫画，描绘了国会的马修·里昂和罗杰·格里斯沃尔德之间的争斗。里昂后来依据《防治煽乱法》被起诉。

[Source: Courtesy, Library ofCongress]

佛蒙特州众议院的共和党议员马修·里昂曾经给《佛蒙特纪闻报》的编辑写信，说他反对一个“恋栈、浮夸、逢迎、贪婪”的政府。仅仅因为这个评价，联邦最高法院的大法官威廉·帕特森在地方赶场的时候恰好成为初审法官。帕特森判决该议员四个月的监禁，1000美元的罚金。这个明显有失公允的判决，激起了当地人民的强烈不满，结果还在监禁中的里昂，却依然高票连任了议员席位。当他后来出狱返回议会时，受到了全体议员长时间的起立鼓掌欢迎。

1798年，喜欢到处旅行的演说家戴维·布朗在马萨诸塞州停留时，发表了一番谴责政府参与土地投机活动的演讲，另外他还批评了《防治煽乱法》。没想到，现场亢奋异常的听众中有人打出了一条标语：“终结美国暴政，总统和平下台，副总统和在野党万岁，愿美德成为政府之基。”就因为这条标语，布朗身陷牢狱。最高法院塞缪尔·蔡斯（SamuelChase）法官也是在赶场的时候，巡回到了马萨诸塞。他以扰乱社会秩序为由判处布朗入狱十八个月，罚金480元。到1800年12月时，布朗已服刑期满，却因为交不起罚金而只得继续坐牢，直到3个月后，新总统杰佛逊赦免所有《防治煽乱罪》的入狱者。

然而这些都还不是整个防治煽乱罪中存续时期内最臭名昭著的案例。1800年，评论家詹姆斯·卡伦德出版了一个单行本小册子。在书里面，他严厉的抨击现任总统亚当斯，说在“亚当斯和杰佛逊之间做选择，就等于在战争与和平、贫穷与富足之间做选择。”这本书招致了诉讼，这次仍然是大法官蔡斯主审，而他此次的表现更加的有问题，完全有亏法官的中立性。在庭审中，蔡斯法官不仅苛刻的问责辩护律师，导致对方竟然中途愤而离场，而且引导全部由联邦党人组成的陪审团做出有罪判决：被告入狱九个月，罚金200元。可没想到的是，卡伦德是个不轻易认怂的家伙，你关的住我的人，关不住我的声音。在狱中时恰逢下一次的总统大选，卡伦德奋笔疾书，继续投书媒体，直陈蔡斯法官的种种卑劣行径。这些文章在竞选的关键时候给联邦党带来了巨大的负面冲击。

塞缪尔·蔡斯（Samuel Chase）

[Source: Maryland State House,Senate Chamber]

根据统计，从1798年7月颁布《防治煽乱法》开始到1801年3月该法案有效期满自然终止，不到3年的时间内，共有十四个人因为该法入狱，而且主要集中在总统大选投票前的1800年。《防治煽乱法》的政治工具的属性没法再更明显了，民心几乎一边倒的开始反对这一恶法。

共和党人虽然占据国会少数地位，但他们从一开始就在抵制《防治煽乱法》。最初的时候就有共和党议员提出《防治煽乱法》事实上是违宪的，因为它公然践踏了宪法第一修正案之言论自由条款。对此，联邦党人马上搬出了看家法宝，英国宗师布莱克斯通对于言论自由的解释。他们说防治煽乱法并没有限制出版的事前自由，只是在事后根据言论带来的后果进行追究，所以没有违宪的问题。历史的前进就要看谁能够提出比布莱克斯通更有说服力的新理念了。有一个人做到了，那就是麦迪逊。实际上，麦迪逊和杰佛逊两人早就在为反对《防治煽乱法》而努力，但是在当时恶法在侧的险境中，为了避免授人以柄，反遭指控，他们只好秘密联系。随着民怨的逐渐升高，时机渐渐成熟。杰佛逊为肯塔基州议会起草了一份重新审议《防治煽乱法》的草案，但更重要的还是后面麦迪逊为维吉尼亚州起草的决议。在这份决议中，麦迪逊对《防治煽乱法》给出了如下的评价：

“行使了宪法从未授予的权力，而且，恰恰还是宪法明确禁止的权力。与其他权力相比，这些权力尤其值得公众的警惕，因为它不仅限制着人们自由检视公众人物和公共事务的权利，还限制着人们自由沟通的权利。更为重要的是，以上的这些权利是维系人民其他权利的唯一有效的保障。”

麦迪逊的论述铿锵有力，他直指布莱克斯通解释的软肋。所谓英格兰式的自由言论观，之所以仅仅强调事前的不审查而不承诺事后的不担责，乃是因为王权理念的作祟。正如维吉尼亚州众议员约翰·尼古拉斯所说“在英国，世袭的君主被视为不可侵犯的神，政府官员则是神的代理人”，所以他们不可以遭到任何形式的诋毁。然而在美国，所有的官员皆是人民的公仆，是经由选举产生的被授权人。自然的，对这些人来说，人民的批评性言论不再是什么攻击，而只是监督他们的行径时的意见表达。对此没有任何可以被追责的理由。

“麦迪逊前提”（the Madisonian premise）成为美国言论自由观念的重要基石

[Source：wikimedia commons]

在维吉尼亚报告中，麦迪逊继续论述说，在美国，是人民而不是政府拥有绝对主权，这和英国有着天壤之别。

“在如此巨大的差异下，难道授予媒体更多的自由，不是合理而且必要的吗？媒体针砭公共人物和公众事务的权力，连最严苛的普通法都未曾禁止。出版自由是其他一切自由的基础，可是美国却连这一基础也没有打好。”

麦迪逊论述的最大价值在于他鲜明的点出了自由检视公众人物和公共事务是人们的基础性权力，是其他合法权利的唯一前提。从此，这个说法就被称作麦迪逊前提，成为影响美国言论自由价值观数十年的重要理念。不过，论争从来都不是很简单就会分出输赢的。在麦迪逊的报告出炉之后，维吉尼亚的联邦党议员们也发表了一份少数派声明，继续支持《防治煽乱法》。他们当然无法挑战主权在民的共识，故此这些议员只能从另一个角度来反驳麦迪逊前提：

“如果无权处罚攻讦政府的言论，我们的国家就无力维持稳定，无力保护自己免遭“邪恶公民”的不断侵扰。政府乃是为维护人们的安全和幸福而存在，与人民的利益紧密相关。既然如此，为了确保政府的安全，人民应当允许政府立法惩治造谣惑众，肆意诽谤的行为，因为这些行为将导致人民对政府的疏离和不信任。”

简单的来说，少数派报告和麦迪逊前提的分歧就在于批评本身，哪怕是激烈的批评会不会就会导致国家的动摇。麦迪逊认为这本身就是人民行使主权的标志，汰换错谬不会引起什么问题而只会纠正政府的过失，而少数派则觉得这些混乱足以颠覆政府，并后续影响到人民生活的秩序。

不管双方的理念如何针锋相对，美国民众的心中对于《防治煽乱法》早已心有定见，这直接表现在新一届美国总统的选举结果上。反对《防治煽乱法》的共和党候选人杰佛逊不仅强势胜出，而且在国会里还彻底打垮了联邦党的政治势力，令其不久之后渐渐消散。上台后的杰佛逊立即宣布特赦所有因《防治煽乱法》而入狱服刑的人，这部法律也因为时效已到自然终结。杰佛逊在写给前第一夫人的信中这样解释自己的特赦：

“我释放了所有因《防治煽乱法》而被关押、起诉的人，因为我始终认为，而且现在也还这样认为，这部法律根本是无效的。它就好比国会命令我们匍匐在地，对着一个金质偶像膜拜，并且让我们时刻监视着，揪出那些不愿从命的人。而我现在所做的，正是把那些个拒绝崇拜偶像而被丢进火坑的人，迅速的抢救出来。”

《防治煽乱法》因党争而来，可贵的是，获胜的杰佛逊就没有因此而报复，因为言论自由的理念让他懂得什么才是更为重要的价值，在就职演说中，杰佛逊说道：

“我们是共和党人，我们也都是联邦党人。如果我们当中有任何人视图让联邦解体，或者改变共和政体，那就让他们不受任何干扰的畅所欲言吧。容忍错误意见的存在，让不同观点交锋辩论，正是我们得享安全的基础所在。”

一部恶法除去了，新的大旗扬起了，然而什么是言论自由，对待这个问题真的有了明确的答案了吗？未必。因为下一轮的争斗很快又会到来。

杰佛逊和麦迪逊交谈，他们共同反对《防治煽乱法》。
