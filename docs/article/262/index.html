很重视友情却不被对方看重是一种怎样的体验？

Posted on 2020/1/19 by S时代

 

↑点击上方“古代神话杂谈”关注我们


故事，要从一个问题开始讲起


“很重视的友情对方却不看重是一种怎样的体验？”


刚开始 大家都在一本正经的诉着苦

直到这个答案的出现



杜甫为李白写了这么多

而李白却只有一首《赠汪伦》


于是大家纷纷为杜甫打抱不平

李白，你的良心不会痛吗？



后来

有忽明真相的吃瓜群众爆料

李白这样做 也是情有可原


…

为何如此悲伤

只怪太过相似

就像

落花有意随流水

流水无心恋落花


不过 在小金看来

古人的“赠某某” 一般多是书信

经人大量传抄了 才会流传开来

所以白童鞋和杜童鞋的故事嘛

应该大致仿佛就是酱婶儿的

…

…

各部门注意：

…

…

Duang——


文豪甲：李白，杜甫来信了吗？

李白：来了十几封呐！

文豪乙：那借我们传抄一下呗？

李白：好啊，杜兄这文采，保证你们越看越喜欢，拿去传抄吧！

…


另一边


文豪乙：杜甫，李白给你回信了吗？

杜甫：嗯呐！写的可好了，比我写给他的好多啦，高兴到炸裂~~

文豪乙：那借我们传抄一下呗？

杜甫：不给！！！

…

 


END

咳咳，小金觉得吧，以现存诗歌的数量来衡量感情，本身就不妥当。


就好像，有的感情轰轰烈烈，有的感情意味深长，有些情绪会体现在字里，有些情绪却珍藏在心里。


但有一点可以肯定：真正的友谊，绝对不会随着时间和空间转换而变质，也不会因对方的喜怒哀乐而改变。
