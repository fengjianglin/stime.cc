《中国科学》杂志社诚聘科学编辑！生命与医学科学方向

Posted on 2020/1/8 by S时代

这里是国内科技期刊出版领域最专业、最具影响力的发展平台之一，这里有结识国内外一流科学家并与他们交流合作的机会。欢迎您加入学术期刊编辑队伍，为中国的学术出版添砖加瓦，作出贡献。

《中国科学》杂志社成立于1995年，编辑出版的期刊包括《中国科学》系列、《科学通报》及《国家科学评论》等。根据业务发展的需要，现诚招生命与医学科学方向科学编辑。

招聘岗位：

科 学 编 辑

生命与医学科学

招聘条件：

  1. 敬业爱岗、善于协作，热爱学术期刊出版事业。

  2. 具有基础医学、分子与细胞生物学、生物物理学等相关学科领域的博士学位。有开阔的科学视野，对科研前沿有敏锐的洞察力。

  3. 年龄在40周岁以下。

  4. 有较强的中英文写作和语言表达能力。

  5. 具备以下条件之一：

  • 有2年以上科研工作经历，发表过高水平科研论文

  • 有2年以上科技期刊编辑工作经历

  • 具有海外留学或工作经历

  • 有高级专业职称

岗位职责：

  1. 追踪科研动态，策划选题，组织发表重要科研成果。

  2. 对学术文章进行初审，并组织专家开展评议。

  3. 对文章进行必要的修改和润色，提高学术成果的可读性。

  4. 撰写热点新闻稿件和宣传文案，促进学术成果传播，提高期刊品牌影响力。

  5. 配合编委会开展工作，完成编委会制定的各项任务。

  6. 完成杂志社和编辑部交办的各项任务。

福利待遇：

  1. 行业内具有竞争力的薪资；六险二金（含企业年金、补充医疗保险）；年度体检。

  2. 优质食堂，提供免费午餐、晚餐。

  3. 丰富多彩的企业文化活动等。

联系方式

有意应聘者请尽快将个人简历（附近期照片一张）电子版发至我公司邮箱。

电话：010-64016350

Email：office@scichina.org

期 刊 简 介

《中国科学》系列和《科学通报》创刊于1950年，由中国科学院和国家自然科学基金委员会共同主办，是我国科技学术期刊中的知名品牌。

《中国科学》系列

按不同学科分设各辑，包含中、英文期刊。致力于各专业领域中研究成果的及时、准确发表，努力与科研人员真诚沟通、提供最专业的服务。
